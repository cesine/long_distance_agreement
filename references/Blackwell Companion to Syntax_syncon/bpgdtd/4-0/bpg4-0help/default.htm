Blackwell Publishing
DTD 4 documentation

This is the documentation website for the Blackwell Publishing XML DTD version 4.0.

How to use this site

There are now five ways of navigating through the DTD and its documentation:

  1. Element index Go directly to any element from the element index
  2. Page images Click on any part of typeset pages to link to the XML element behind the text.
  3. Search Download the HTMLHelp version of this website which allows you to search the entire site. The HTMLHelp version is also preferable for printing as it retains the CSS styling of the on-screen display.
  4. Tree view Dynamically generate a tree view of any element in the DTD and link to each element's help page.
  5. Top-down Choose a top-level element and drill down through its sub elements

The website also includes

  • frequently-asked questions
  • special topics on various aspects of using the DTD
  • sample files: DTD 3 SGML, DTD 4 XML and BEES files

Downloads

  • This website as a zip file This complete web site can be downloaded as a zip file for browsing off-line. Extract the contents to a local directory and begin browsing at default.htm. However, if you do download this documentation, check back frequently to ensure you have the most recent version of the documentation and the DTD. A log of recent changes to the documentation is kept here.
  • This website as a HTML Help file This site can is also available as a Microsoft HTML Help file. This version includes a full text search feature.
  • Download the DTD The current version of the DTD can be downloaded here.

This is the current production version of the DTD. Details of recent changes, the update schedule for possible future changes and procedures for requesting changes to the DTD are available on the Update page.
